# Lab 01 — Azure AD & Identity

## What I Did
- Created and configured Azure AD users (Member + Guest)
- Created Security Groups with assigned membership
- Created a new Azure AD tenant
- Managed B2B Guest user invitations

## Key Learnings
- Guest users show as #EXT# in username
- Deleted users can be restored within 30 days
- Azure AD Connect syncs on-prem AD every 30 minutes
- Member = internal tenant user | Guest = external B2B user

## Exam Tips
- Conditional Access requires Azure AD Premium P1 minimum
- Azure AD is NOT same as Windows AD — no GPO, no OUs
- One tenant can have unlimited subscriptions
